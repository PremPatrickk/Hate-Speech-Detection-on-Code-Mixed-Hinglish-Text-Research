# Hate-Speech-Detection-on-Code-Mixed-Hinglish-Text-Research
Transformer-based Hate Speech Detection in Hindi-English Code-Mixed Text

This repository contains the implementation and datasets used for my thesis on transformer-based hate speech detection.

## 📊 Datasets
- THAR Dataset
- HASOC Dataset
- AOLD Dataset
- Combined Dataset

## ⚙️ Models Used
- Multilingual: mBERT, XLM-R, IndicBERT, MuRIL
- Code-Mixed: HingBERT, HingRoBERTa
- LLMs: LLaMA-3, Mistral (Zero-shot)

## 🧪 Experiments
- Data preprocessing and standardization
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Performance comparison using F1-score
